Gym Members Exercise Analysis 
An end-to-end data analysis project exploring how age and other member attributes relate to workout behavior, built with Python, SQL, and machine learning.
Project Overview
This project investigates a central question: does age meaningfully influence workout behavior and performance?

Using a Kaggle dataset of gym member exercise tracking data, the analysis moves through a complete data science workflow: data cleaning, SQL-based querying, exploratory data analysis, and predictive modeling. When EDA revealed that age had a negligible correlation with workout metrics, the project pivoted to a more productive question: what features actually drive workout intensity?
 Dataset
Source: Gym Members Exercise Tracking Dataset (Kaggle)
Features include: age, gender, weight, height, heart rate metrics (max/avg/resting BPM), session duration, calories burned, workout type, body fat percentage, water intake, workout frequency, experience level, and BMI
Tech Stack
Python — pandas, NumPy
SQL — SQLite via sqlite3
Visualization — Matplotlib, Seaborn
Machine Learning — scikit-learn (Random Forest)
 Methodology
1. Data Collection & Cleaning
Validated dataset structure, checked for missing values and duplicate rows
Detected outliers across all numerical columns using the IQR method
Visualized outliers with boxplots (weight, calories burned, BMI)
Replaced outliers with category-level means (grouped by workout type) to stabilize downstream modeling
Converted metric units (kg → lbs, m → ft) for readability
2. SQL Analysis
Loaded the cleaned data into a SQLite database and answered analytical questions directly in SQL, including:

BMI distribution and variance across all members
BMI trends grouped by gender (average, min, max)
Top members by calories burned
Average session duration by experience level
Highest calorie-burning workout type
Body fat percentage for members training 3+ days/week
Water intake vs. calories burned relationship
3. Exploratory Data Analysis
Age distribution histogram with KDE
Age vs. calories burned scatterplot with regression line
Average workout frequency by age group
Stacked bar analysis of workout frequency by age group and experience level
Full correlation matrix heatmap across all numerical variables

Key EDA insight: age showed weak correlation with essentially all workout metrics — motivating a pivot from age-focused analysis to feature importance modeling.
4. Machine Learning
Trained a Random Forest classifier to predict workout intensity from age, experience level, workout frequency, session duration, and BMI
Evaluated performance with a classification report (precision, recall, F1)
Extracted and visualized feature importances
Key Findings
Age is not a strong driver of workout behavior. Correlations between age and workout metrics (frequency, duration, calories burned) were negligible.
Session duration dominates workout intensity prediction, accounting for ~50% of feature importance — far ahead of BMI (~17%), age (~14%), and experience level (~13%).
Workout frequency contributed the least (~6%) to predicting intensity, suggesting that how long someone trains per session matters more than how often they show up.
 How to Run
Clone this repository:

git clone https://github.com/<your-username>/gym-members-analysis.git

cd gym-members-analysis

Install dependencies:

pip install pandas numpy matplotlib seaborn scikit-learn

Download gym_members_exercise_tracking.csv from the Kaggle link above and place it in the project root.
Launch the notebook:

jupyter notebook my_project.ipynb
